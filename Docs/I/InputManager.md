## 1. Usage
The Input Manager is initialized at runtime by the engine. (see [IsoGame.cs](/IsoGame.cs))

It handles inputs from the user and relays the relevant information via events to the game.
The game needs to register actions and their associated keybinds for the engine to consider sending an event. (See [Action Mappings](#2-action-mappings) & [Axis Mappings](#3-axis-mappings))
