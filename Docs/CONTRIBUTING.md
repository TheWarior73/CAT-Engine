# Contributing To CAT Engine

## 1.
Contributing is encouraged, however it needs to follow some rules :

- Use `camelCaseNotation` for variables - First letter in lowercase, then each new word's first letter in uppercase
- Use `PascalCase` for functions - First letter of each word is uppercase
- Don't use `m_...` for class member
- Comment your code. If not, it needs to be understood with a quick glance.

## 2. How to contribute 
Fork the project, work, commit etc...
Once you are satisfied with your additions, submit a pull request.
Make sure your request is easily understandable, that it lists all that you have added, and if it solves some Issues, cite them as well !
